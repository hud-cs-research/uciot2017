<h1 style="display: inline-block">How to host your HTML website on GitHub Pages for free</h1>
</header>
<div class="entry-meta hidden-print">
<p style="display: inline-block">Categories: <a href="http://docs.pinegrow.com/category/misc">Misc</a> &#124; Tags: <a href="http://docs.pinegrow.com/topic/hosting">Hosting</a></p>
</div>
<main class="entry-content">
<div class="lead" style="display: inline-block">
<p>GitHub is a great place for hosting your HTML websites. It&#8217;s free and you can even use your own domain names.</p>
</div>
<blockquote><p>BTW. Our main website Pinegrow.com is a static HTML website hosted on GitHub Pages.</p></blockquote>
<h2><span id="Sign-up_for_GitHub">Sign-up for GitHub</span></h2>
<p><a href="https://github.com/" target="_blank">Create a free GitHub account.</a> Skip this step if you already have one.</p>
<h2><span id="Download_GitHub_Desktop_app">Download GitHub Desktop app</span></h2>
<p><a href="https://desktop.github.com/" target="_blank">Download GitHub Desktop application for Mac or Windows.</a> The app will let you work with git through GUI. Skip this step if you already have the app or prefer to use git command line tools.</p>
<h2><span id="Create_a_Git_repository">Create a Git repository</span></h2>
<p>Open <strong>GitHub Desktop</strong> app and click on <strong>Add repository</strong>. Select the folder where your website is located and click on &#8220;Create &amp; Add Repository&#8221; button.</p>
<p><img class="alignnone size-full wp-image-1291" src="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp1.jpg" alt="ghp1" width="471" height="211" srcset="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp1.jpg 471w, http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp1-300x134.jpg 300w" sizes="(max-width: 471px) 100vw, 471px" /></p>
<h2><span id="Publish_therepository_to_GitHub">Publish the repository to GitHub</span></h2>
<p>Type a commit message and press <strong>Commit &amp; Sync master</strong>:</p>
<p><img class="alignnone size-full wp-image-1294" src="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp4.jpg" alt="ghp4" width="983" height="665" srcset="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp4.jpg 983w, http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp4-300x203.jpg 300w, http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp4-768x520.jpg 768w" sizes="(max-width: 983px) 100vw, 983px" /></p>
<p>A dialog will popup. Enter the name of the repository (that will be a part of the website&#8217;s URL), select if you want to keep the repository private (the website will still be public) and press &#8220;Publish Repository&#8221;:</p>
<p><img class="alignnone size-full wp-image-1295" src="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp5.jpg" alt="ghp5" width="628" height="371" srcset="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp5.jpg 628w, http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp5-300x177.jpg 300w" sizes="(max-width: 628px) 100vw, 628px" /></p>
<h2><span id="Create_gh-pages_branch">Create gh-pages branch</span></h2>
<p>The repository is on GitHub but the website is not yet on GitHub Pages. To do that, create a branch named <strong>&#8220;gh-pages&#8221;</strong> (the name tells GitHub to publish this branch as a website) and click &#8220;Create Branch&#8221;.</p>
<p><img class="alignnone size-full wp-image-1296" src="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp6.jpg" alt="ghp6" width="442" height="235" srcset="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp6.jpg 442w, http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp6-300x160.jpg 300w" sizes="(max-width: 442px) 100vw, 442px" /></p>
<p>Now, make sure that <strong>gh-pages</strong> branch is selected and publish the branch (send it to GitHub):</p>
<p><img class="alignnone size-full wp-image-1297" src="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp7.jpg" alt="ghp7" width="740" height="278" srcset="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp7.jpg 740w, http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp7-300x113.jpg 300w" sizes="(max-width: 740px) 100vw, 740px" /></p>
<h2><span id="The_website_is_online">The website is online!</span></h2>
<p>The url of your website will be http://&lt;your-github-username&gt;/github.io/&lt;your-repository-name&gt;. You can even use <a href="https://help.github.com/articles/using-a-custom-domain-with-github-pages/">your own domain</a>, so that the url will be http://&lt;your-domain&gt;</p>
<p><img class="alignnone size-full wp-image-1298" src="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp8.jpg" alt="ghp8" width="525" height="280" srcset="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp8.jpg 525w, http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp8-300x160.jpg 300w" sizes="(max-width: 525px) 100vw, 525px" /></p>
<h2><span id="Publishing_changes">Publishing changes</span></h2>
<blockquote><p>Important: the selected branch should always be <strong>gh-pages</strong></p></blockquote>
<p>To publish changes made to the local copy of your website, open GitHub Desktop app, select your repository, type a commit message and click on &#8220;<strong>Commit and Sync gh-pages</strong>&#8220;:</p>
<p><img class="alignnone size-full wp-image-1300" src="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp14-1.jpg" alt="ghp14" width="965" height="510" srcset="http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp14-1.jpg 965w, http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp14-1-300x159.jpg 300w, http://11115-presscdn-0-17.pagely.netdna-cdn.com/wp-content/uploads/2016/08/ghp14-1-768x406.jpg 768w" sizes="(max-width: 965px) 100vw, 965px" /></p>
